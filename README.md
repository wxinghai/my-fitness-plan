# My Fitness Plan

这个仓库用来按日期归档训练计划。

## 目录规则

- 每一轮正式计划使用一个新的日期文件夹
- 文件夹命名格式：`YYYYMMDD`
- 下一轮计划不要覆盖旧文件，直接新建新的日期文件夹

## 当前结构

  - `背部训练计划.md`
  - `腿部训练计划.md`
  - `胸部训练计划.md`
  - `极限测试计划.md`
  - `下周期设计说明.md`

## 文件说明

- `背部训练计划.md`：当前背日正式执行版
- `腿部训练计划.md`：当前腿日正式执行版
- `胸部训练计划.md`：当前胸日正式执行版
- `极限测试计划.md`：Deload 结束后的测试安排
- `下周期设计说明.md`：给未来继续设计下一轮周期时使用的交接说明

## 更新方式

每次新周期开始时：

1. 保留旧日期文件夹
2. 新建当天日期文件夹
3. 在新文件夹中放入新的背 / 腿 / 胸计划
4. 如有测试安排和下周期说明，也一并放入该日期文件夹

这样可以完整保留每一轮训练计划的演变过程。
