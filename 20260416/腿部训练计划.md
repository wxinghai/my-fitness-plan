# 腿部训练计划

版本日期：2026-04-16  
当前阶段：从循环 5 开始执行  
训练节奏：背 / 腿 / 胸，练 3 天休 2 天，然后重复

## 动作顺序

1. 深蹲
2. 罗马尼亚硬拉
3. 腿部辅助动作按恢复情况安排

## 主项与次主项

| 循环 | 深蹲（第一主项） | 罗马尼亚硬拉（第二主项） |
| --- | --- | --- |
| 5 | `120 kg x 5 x 5` | `95 kg x 3 x 8` |
| 6 | `125 kg x 5 x 5` | `100 kg x 3 x 7` |
| 7 | `130 kg x 5 x 4` | `105 kg x 3 x 6` |
| 8 | `135 kg x 5 x 3-4` | `110 kg x 3 x 6` |
| 9 | `135 kg x 4 x 3 + 140 kg x 1` | `90 kg x 3 x 6` |
| 10 | `140 kg x 3 x 2 + 145 kg x 1` | `85 kg x 3 x 6` |
| 11 | `145 kg x 2 x 2 + 150 kg x 1（RPE9）` | `80 kg x 2 x 6` |
| 12（Deload） | `90 kg x 3 x 5` | 可不做，或 `60-70 kg x 2 x 6` |

## 辅助动作建议

### 循环 5-8

- 保加利亚分腿蹲或腿举：`2-3 x 8-12`
- 腿弯举：`3-4 x 10-15`
- 45 度挺身或反向挺身：`2-3 x 10-15`
- 核心动作：`3-4` 组

### 循环 9-11

- 保加利亚分腿蹲或腿举：`2 x 8-10`
- 腿弯举：`2-3 x 10-15`
- 45 度挺身或反向挺身：`2 x 10-12`
- 核心动作：`3` 组

### 循环 12（Deload）

- 只保留很轻的腿弯举或核心激活

## 执行要求

- 深蹲后期单次暴露以保持动作和神经兴奋为主，不要硬顶到失败。
- 罗马尼亚硬拉在循环 9-11 主动降量，优先服务深蹲恢复。
- 辅助项以后链和稳定为先，不在后期刷辅助项 PR。
