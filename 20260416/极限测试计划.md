# 极限测试计划

版本日期：2026-04-16  
适用阶段：循环 12（Deload）结束后执行  
目标：测试当前卧推、深蹲与背部主项表现

## 总体原则

- 测试日优先追求成功率，不追求无意义的失败尝试。
- 前面的热身和过渡单次尽量节省体力，把最好的状态留给最后 1-2 把。
- 85% 以上的单次不要做太多。
- 每一把上台前都以“动作干净、速度可控”为优先。
- 背部两个主项采用不同测试方式：
  - 负重引体测试单次表现
  - 高位单臂划船测试严格标准下的表现重量，不做绝对 1RM

## 背部主项测试

### 负重引体测试

#### 热身与尝试顺序

1. 体重 `x 5`
2. `+10 kg x 3`
3. `+20 kg x 2`
4. `+30 kg x 1`
5. `+40 kg x 1`
6. `+47.5 kg x 1`
7. `+55 kg x 1`
8. `+60 kg x 1`
9. 视状态决定：`+62.5 kg x 1` 或 `+65 kg x 1`

#### 动作标准

- 每组从干净悬垂开始
- 尽量避免摆动借力
- 至少下巴过杠，最好上胸主动找杠

#### 临场判断

- 如果 `+47.5 kg` 就明显变慢，后面保守推进。
- 如果 `+55 kg` 很稳，`+60 kg` 正常上。
- 如果 `+60 kg` 完成得很顺，最后一把优先冲 `+62.5 kg`，状态非常好再考虑 `+65 kg`。
- 如果 `+60 kg` 已经 grind，就不要为了数字硬上更高重量。

### 高位单臂划船测试

这里不建议做绝对 1RM。  
更建议测试“严格标准下的工作重量表现”，这样对后续周期更有参考价值。

#### 推荐测试方式

- 目标测试：严格标准 `6RM`

#### 热身与尝试顺序

1. `30 kg x 8`
2. `40 kg x 5`
3. `50 kg x 3`
4. `55 kg x 1`
5. `60 kg x 1`
6. `62.5 kg x 6`
7. 如果 `62.5 kg x 6` 很稳，下一把尝试 `65 kg x 6`

#### 动作标准

- 单侧独立完成，左右两边都按同一标准执行
- 肘朝髋走
- 顶峰停顿 `1` 秒
- 下放控制
- 不靠明显躯干旋转和惯性甩起

#### 临场判断

- 如果 `60 kg x 1` 就已经需要明显借力，正式测试不要硬上 `65 kg x 6`。
- 如果 `62.5 kg x 6` 还能保持动作标准，那它就是很好的阶段成绩。
- 如果动作一旦开始明显变形，就停止加重，保留当前最好的一组作为测试结果。

## 卧推测试

### 热身与尝试顺序

1. `20 kg x 10`
2. `40 kg x 5`
3. `60 kg x 3`
4. `80 kg x 2`
5. `95 kg x 1`
6. `105 kg x 1`
7. `112.5 kg x 1`
8. `117.5 kg x 1`
9. 视状态决定：`122.5 kg x 1` 或 `125 kg x 1`

### 临场判断

- 如果 `105 kg` 就明显变慢，后面保守推进。
- 如果 `112.5 kg` 很稳，`117.5 kg` 可以正常上。
- 如果 `117.5 kg` 是顺畅完成，最后一把优先冲 `122.5 kg`，状态非常好再考虑 `125 kg`。
- 如果 `117.5 kg` 已经 grind，不建议为了计划硬上 `125 kg`。

## 深蹲测试

### 热身与尝试顺序

1. `20 kg x 8`
2. `60 kg x 5`
3. `80 kg x 3`
4. `110 kg x 2`
5. `130 kg x 1`
6. `145 kg x 1`
7. `155 kg x 1`
8. `165 kg x 1`
9. 视状态决定：`170 kg x 1` 或 `175 kg x 1`

### 临场判断

- 如果 `145 kg` 的速度一般，后面就按保守路线走。
- 如果 `155 kg` 很稳，`165 kg` 正常上。
- 如果 `165 kg` 明显轻快，最后一把可以冲 `170 kg`，状态非常好再考虑 `175 kg`。
- 如果 `165 kg` 已经很吃力，就不要为了目标数字硬冲更高重量。

## 休息时间建议

- 60 kg 以下：按感觉休息
- 60-80% 区间：`2-3` 分钟
- 80% 以上单次：`4-6` 分钟
- 最后两把冲刺：必要时可以休到 `6-8` 分钟

## 测试日执行提醒

- 卧推和深蹲都要完整热身，但不要因为热身做太多额外组。
- 测试日前一晚优先保证睡眠。
- 测试当天按平时习惯吃饭，不临时大改饮食。
- 如果某一把动作明显变形或失去稳定，后面立即转保守路线。

## 成绩记录建议

测试结束后记录：

- 当天体重
- 负重引体最好成绩
- 高位单臂划船最好成绩
- 卧推最好成绩
- 深蹲最好成绩
- 每一把单次的主观体感
- 哪一把最稳，哪一把最接近极限

这样下一轮周期会更容易调整。
