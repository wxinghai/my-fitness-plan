# 背部训练计划

版本日期：2026-04-16  
当前阶段：从循环 5 开始执行  
训练节奏：背 / 腿 / 胸，练 3 天休 2 天，然后重复

## 动作顺序

1. 单臂高位划船
2. 悍马机宽握划船
3. 面拉
4. 站姿卷腹

## 主项与次主项

下面的单臂高位划船重量默认指单侧工作重量。

| 循环 | 单臂高位划船（第一主项，单侧） | 悍马机宽握划船（第二主项） |
| --- | --- | --- |
| 5 | `52.5 kg x 5 x 8` | `3-4 x 10-12` |
| 6 | `55 kg x 5 x 6-8` | `3-4 x 10` |
| 7 | `57.5 kg x 5 x 6` | `4 x 8-10` |

## 厚度与补充动作

### 悍马机宽握划船

- 循环 5-8：`3-4 x 8-12`，保留 `1-2` 次余力
- 循环 9-11：`3 x 8-10`，保留 `1-2` 次余力
- 循环 12（Deload）：`2-3 x 8`

### 面拉

- 循环 5-8：`3 x 15-20`
- 循环 9-11：`2-3 x 15-20`
- 循环 12（Deload）：`2 x 15`

### 站姿卷腹

- 循环 5-8：`4 x 12-15`
- 循环 9-11：`3-4 x 10-12`
- 循环 12（Deload）：`2-3 x 10-12`

## 执行要求

- 单臂高位划船保持“肘朝髋走”，顶峰停顿 1 秒，下放控制。
- 悍马机宽握划船以中上背发力和肩胛后缩为主，不追求乱甩重量。
- 卷腹以骨盆主动后倾和腹部卷起为前提，按目标次数反推重量。
