# 胸部训练计划

版本日期：2026-04-16  
当前阶段：从循环 5 开始执行  
训练节奏：背 / 腿 / 胸，练 3 天休 2 天，然后重复

## 动作顺序

1. 卧推
2. 卧推补充动作
3. 胸肩三头辅助动作

## 主项

| 循环 | 卧推 |
| --- | --- |
| 5 | `90 kg x 5 x 5` |
| 6 | `92.5 kg x 5 x 5` |
| 7 | `95 kg x 5 x 4-5` |
| 8 | `97.5 kg x 5 x 3-4` |
| 9 | `100 kg x 4 x 3 + 105 kg x 1` |
| 10 | `102.5 kg x 3 x 2 + 107.5 kg x 1` |
| 11 | `105 kg x 2 x 2 + 110 kg x 1（RPE9）` |
| 12（Deload） | `65 kg x 3 x 5` |

## 补充与辅助

### 卧推补充动作

- 暂停卧推或窄握卧推：`3-4 x 4-6`

### 辅助动作建议

#### 循环 5-8

- 上斜哑铃卧推：`2-4 x 6-10`
- 三头下压或仰卧臂屈伸：`3-5 x 8-15`
- 侧平举：`3-4 x 12-20`
- 面拉或肩袖小肌群：`2-3 x 15-20`

#### 循环 9-11

- 上斜哑铃卧推：`2-3 x 6-10`
- 三头下压或仰卧臂屈伸：`2-4 x 8-15`
- 侧平举：`2-3 x 12-20`
- 面拉或肩袖小肌群：`2 x 15-20`

#### 循环 12（Deload）

- 只保留很轻的肩部和肩袖激活

## 执行要求

- 卧推后期单次暴露优先保证动作和速度，不为计划硬顶失败。
- 如果循环 5-7 开始明显停滞，可优先保留暂停卧推作为技术补充。
- 后期辅助项主动降量，把恢复资源优先留给卧推主项。
